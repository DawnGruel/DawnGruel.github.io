# 公司内部文件共享方法
1.新建一个需要共享的文件夹
2.右击文件夹，选择属性选项
3.选择属性选项卡下的“共享”选项卡，并点击“共享”按钮
4.在打开的“网络访问”界面中输入“Everyone”用户，点击“添加”后点击“共享”
5.查看本地IP地址

# 查看方式
## windows
Win+R 在弹出的运行命令框中输入明莉格式为“\\(共享主机IP地址)”的访问命令
## mac
打开访达，顶部选择“前往-链接服务器”
在输入框中输入“smb://(共享主机IP地址)”

::: tip 备注
步骤4添加的“Everyone”用户代表任何用户都可访问，可根据实际情况设置只读或读写权限。
:::
